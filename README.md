Steps to start frontend on your machine:
- clone this branch ( git clone https://github.com/airavata-courses/Zilean.git)
- go to frotnend folder (cd frontend)
- install all the dependencies from the package.json file (npm install)
- after successfully installing the dependencies, run this command "npm run dev" and this will start a local development server on port 3000
- visit http://localhost:3000/ , frontend should be up and running
